Binary Search is an efficient searching algorithm that works on sorted arrays. Instead of checking each element sequentially, it divides the search interval in half repeatedly, reducing the number of comparisons significantly.

  1. Start with low at the start and high at the end of the array.

  4. If arr[mid] < target â†’ search the right half.
  5. If arr[mid] > target â†’ search the left half.
  6. Repeat until low > high.

function binarySearch(arr, target):
    low = 0
    high = length(arr) - 1

    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        else if arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1

    return -1
